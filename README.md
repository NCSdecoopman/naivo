# naivo.fr — Assistant de ski

naivo.fr est un outil qui centralise sur une carte les informations essentielles pour préparer une sortie en montagne.

- le risque avalanche officiel (BERA – Météo-France)
- les stations météo avec des relevés horaires (neige, température, vent)
- les webcams,
- les conditions et sorties Skitour
- les prévisions météo détaillées (meteogrammes)

L’objectif est de gagner du temps, améliorer la lecture globale des conditions et aider à la prise de décision avant le départ, en rendant les données plus accessibles et cohérentes spatialement.

naivo ne remplace ni l’analyse terrain ni le jugement du pratiquant.
Il s’inscrit comme un outil complémentaire, conçu pour structurer l’information, réduire la dispersion des sources et faciliter une préparation plus éclairée.

Accessible gratuitement sur https://naivo.fr
