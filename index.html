À propos

Sources des données

Ce site rassemble et organise plusieurs types d’informations météorologiques et nivologiques provenant de sources fiables, reconnues et complémentaires.

Météo-France

Les données d’observation issues des stations météorologiques ainsi que les Bulletins d’Estimation du Risque d’Avalanche (BERA) proviennent de Météo-France. Elles sont obtenues via les API officielles :

  • API Paquet Observations
    Documentation
  • API Bulletin Avalanche (BERA)
    Documentation

Ces données offrent une base solide pour analyser les conditions météorologiques, nivologiques et d’enneigement, ainsi que la stabilité du manteau neigeux.

Webcams SpotAiR

Les images des webcams proviennent du réseau SpotAiR, via l’API officielle :

  • Documentation

SpotAiR met à disposition un ensemble de webcams de montagne et d’activités outdoor. L’accès est gratuit mais nécessite l’obtention préalable d’une clef API personnelle.

meteoblue

Les prévisions météorologiques sont fournies par meteoblue, notamment sous forme de météogrammes, d’images et de données numériques haute résolution :

  • Image API (météogrammes, cartes, graphiques)
    Documentation

Ces prévisions complètent les observations de terrain et offrent une vision prospective indispensable à la préparation d’une sortie en montagne.

Engagement et transparence

L’objectif du site est de centraliser des informations provenant de plusieurs fournisseurs pour offrir une vue claire, cohérente et facilement exploitable. Chaque donnée affichée conserve la traçabilité de sa source, et les services externes sont utilisés dans le respect de leurs conditions d’utilisation.

Remarque

Toute sortie en montagne comporte des risques : les informations fournies sur ce site ne remplacent ni l’analyse personnelle des conditions, ni l’expérience, ni la prise de décision sur le terrain. L’utilisateur reste seul responsable de la préparation et de la conduite de ses activités.

Ajouter Naivo à l’écran d’accueil (iOS)

Sur iPhone / iPad (Safari)

Pour installer Naivo comme une application sur votre écran d’accueil :

  1. Ouvrez Naivo dans Safari.
  2. Touchez le bouton Partager (carré avec une flèche vers le haut).
  3. Faites défiler la liste des actions.
  4. Sélectionnez « Sur l’écran d’accueil ».
  5. Validez le nom de l’app puis touchez Ajouter.

Remarque

Cette opération doit être faite directement depuis Safari. Les autres navigateurs iOS (Chrome, Firefox, etc.) ne permettent pas toujours l’installation PWA avec toutes les fonctionnalités.
